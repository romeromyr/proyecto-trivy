# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.70.x  | :white_check_mark: |
| 0.69.x  | :white_check_mark: |
| < 0.69  | :x:                |

## Reporting a Vulnerability

Please report security vulnerabilities to security@aquasec.com.

## Security Scanning

This repository uses automated security scanning via GitHub Actions and third-party tools.

## Supply Chain Security

All release artifacts are signed using Cosign and Sigstore. Verify signatures before use.
